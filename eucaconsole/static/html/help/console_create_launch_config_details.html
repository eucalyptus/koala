Create Launch Configuration

Details

This panel allows you to specify the name, instance size, and user data associated with the launch configuration.

  1. Type the name of your launch configuration in the Name text box.
  2. Select an instance size/type from the Instance type drop-down list.
  3. Specify custom user data by typing it into the User data text box or by attaching a file by clicking the Choose file button.
  4. Click the Next button to proceed to the Security tab.
